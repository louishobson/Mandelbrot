# Mandelbrot #

Employs GLFW and OpenGL to render an interactive Mandelbrot set

## Supported Platforms ##

* Any Linux-based OS

## Installation ##

### Dependancies ###

* GLFW
* OpenGL

### Installation ###

```
git clone https://github.com/louishobson/Mandelbrot.git
cd Mandelbrot
make
make install
```

### Uninstallation ###

```
make uninstall
make revert
```

## Running the Program ##

```
mandelbrot
```
