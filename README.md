# Mandelbrot #

Employs GLFW and OpenGL to render an interactive Mandelbrot set

## Supported Platforms ##

* Any Linux-based OS

## Installation ##

### Dependancies ###

* GLFW
* OpenGL

### Building from Source ###

```
git clone https://github.com/louishobson/Mandelbrot.git
cd Mandelbrot
make mandelbrot
```

## Running the Program ##

```
./mandelbrot
```
